# Six Omens

A tense narrative horror game made for the Jamsepticeye game jam.

Answer the phone. Read the omens. Decide: will they live or die?

**[Play on itch.io](https://cannolicat.itch.io/six-omens)**

## About

Six Omens is a short horror game built in Unity. Each round you answer a phone call and are shown a set of hidden omen cards. Skulls mean death, halos mean life. Read the odds, use your tools, and make your guess before the timer hits zero. Correct guesses earn money. Wrong ones cost it. Go bankrupt and it's over.

## Features

- Timed decision rounds with a streak/speed multiplier economy
- Item system with consumable and passive items (Stopwatch, Magnifying Glass, Hourglass, X-Ray Glasses, Sleight of Hand)
- Peek and flip mechanics that trade time for information
- Crossfading music system 

## Built With

- Unity (URP, UI Toolkit)
- C#
